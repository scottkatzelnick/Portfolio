About Password Generator

Author: Scott Katzelnick

Developed using the Django Framework®

™

This app serves as an introductory assignment to learning how to develop a website/webapp using the Django Framework®-
a comprehensive, fully accessible Python framework that makes it easy for anyone to get started creating their own web projects.

Specifically, this app uses a little bit of HTML, CSS, JavaScript and Python logic to allow a user to choose from a couple options
such as the length of the password desired, if it should include uppercase letters, special characters, etc. The app is simple and easy
to navigate thanks to a few clearly labled buttons that take you to the desired pages.

Feel free to use this app freely and openly to generate secure, un-guessable passwords for your password protected accounts.
Just make sure to store them in a safe place so as not to lose them forever, if you're forgetful.
